# Журнал змін

Усі значні зміни в цьому проекті документуються тут.

## [2.0.0] — 2025-02-21

### Додано
- **Три режими доступу**: переглядач (без авторизації), редактор (пароль), адміністратор (окремий пароль)
- **Адмін-панель** (`admin.html`) з темною темою для налаштування додатку
- Можливість встановлювати назву класу та школи через адмін-панель
- Управління паролем редактора через адмін-панель (збереження в Firebase)
- Базовий розклад для автозаповнення (редагується через адмін-панель)
- Поле для збереження конфігурації Firebase в адмін-панелі
- **Швидке додавання ДЗ ("Сьогодні задали")** — кнопка для миттєвого призначення завдань на наступний такий самий урок.
- **Drag & Drop** — зміна порядку уроків перетягуванням в режимі редактора
- Окремий файл `js/firebase-config.js` для конфігурації Firebase
- Окремий файл `js/admin-credentials.js` для пароля адміністратора
- Документація по розгортанню на GitHub Pages (`docs/github-pages-deploy.md`)
- Документація по інтеграції з Firebase (`docs/firebase-setup.md`)
- Файл ліцензії MIT (`LICENSE`)
- Цей файл журналу змін (`CHANGELOG.md`)

### Змінено
- **[ОПТИМІЗАЦІЯ]** Кардинально зменшено навантаження на базу даних Firebase (з 44 запитів до 2 ефективних діапазонних запитів при завантаженні).
- Головна сторінка тепер відображає назву класу та школи з Firebase
- Модальне вікно входу з вибором ролі (Редактор / Адміністратор)
- Оновлено структуру проекту для простішого налаштування
- Покращено адаптивність кнопок у шапці для планшетів та смартфонів
- **[КЕШУВАННЯ]** Додано систему версіонування кешу (v2.0.0), яка автоматично очищує старі несумісні дані з телефонів учнів при першому заході на нову версію, запобігаючи помилкам.

### Виправлено
- Виправлено критичну помилку збереження предметів, що містять апостроф (наприклад, "Основи здоров'я").
- Виправлено баг, через який розклад дублювався в базі даних при видаленні домашнього завдання (додано автоматичне очищення бази).
- Виправлено баг з видаленням домашнього завдання через форму "Сьогодні задали" (тепер пусті рядки коректно видаляють запис з бази).

### Видалено
- Захардкоджені облікові дані (логін/пароль) з основного скрипту
- Інлайн конфігурація Firebase з `app.js`
