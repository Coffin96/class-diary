# 📚 Електронний щоденник 5-Б класу

Простий веб-застосунок для перегляду домашніх завдань.

## 🚀 Для учнів
1. Відкрийте сайт
2. Переглядайте розклад та домашні завдання
3. Переключайте тижні кнопками ← →

## 🔑 Для старости
**Логін:** `starosta`  
**Пароль:** `class123`

1. Натисніть "Режим редагування"
2. Введіть логін та пароль
3. Редагуйте уроки та завдання
4. Зміни зберігаються автоматично
5. Натисніть "Вийти" після редагування

## ⚙️ Налаштування Firebase

1. Створіть проект на [Firebase Console](https://console.firebase.google.com/)
2. Увімкніть **Realtime Database** (Test mode)
3. Скопіюйте конфігурацію у файл `js/app.js`
4. Встановіть правила безпеки (див. нижче)

### Firebase Security Rules
```json
{
  "rules": {
    "schedule": {
      ".read": true,
      ".write": true
    },
    "homework": {
      ".read": true,
      ".write": true
    }
  }
}
```
