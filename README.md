# 📚 Електронний щоденник класу

Веб-застосунок для перегляду розкладу та домашніх завдань. Працює на GitHub Pages + Firebase Realtime Database.

## ✨ Можливості

- 📖 **Перегляд** розкладу та домашніх завдань без авторизації
- ✏️ **Редагування** уроків та завдань (режим редактора)
- ⚙️ **Адмін-панель** для налаштування додатку
- 🔀 **Drag & Drop** зміна порядку уроків
- ⚡ **Швидке додавання** ("Сьогодні задали") з автоматичним пошуком наступного такого ж уроку
- 📱 Адаптивний дизайн для телефонів та планшетів
- 💾 Автозбереження в Firebase + резервна копія в localStorage
- 🚀 **Оптимізована робота з мережею** (мінімальне навантаження на безкоштовні ліміти Firebase)

## 🏗️ Три режими роботи

| Режим | Доступ | Опис |
|-------|--------|------|
| Переглядач | Без пароля | Перегляд розкладу та домашніх завдань |
| Редактор | Пароль | Додавання/редагування уроків, домашніх завдань |
| Адміністратор | Пароль | Налаштування класу, школи, паролів, базового розкладу |

## 🚀 Швидкий старт

1. Налаштуйте Firebase → [Інструкція](docs/firebase-setup.md)
2. Розгорніть на GitHub Pages → [Інструкція](docs/github-pages-deploy.md)
3. Відкрийте `admin.html` для первинного налаштування

## ⚙️ Налаштування

### Пароль адміністратора
Файл `js/admin-credentials.js`:
```javascript
const ADMIN_PASSWORD = 'ваш_пароль';
```

### Конфігурація Firebase
Файл `js/firebase-config.js` — вставте свої дані з [Firebase Console](https://console.firebase.google.com/).

### Пароль редактора
Встановлюється через адмін-панель (`admin.html`). За замовчуванням: `edit123`
